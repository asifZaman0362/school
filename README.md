# School
---
This repository is a collection of projects/assignments I took on while in college
trying to learn how computers and related technologies work. This repository is a 
compilation of submodules to the seperate projects organised and categorised in the
most logical way I could think of *(I am not very good at tidying things up).*


In its earlier stages, this repo used to be a horror story where everything was 
pushed to this one single repo which made it absolutely impossible to manage. 
However, one lousy evening I decided to break things up into smaller chunks, put them
in their own seperate repos and add them to this <main> repo as submodules.


I hope to never have to cringe at myself when I look at this mess ten years from now
when I've (hopefully) grown up and finally learned to keep things **properly** organised.
